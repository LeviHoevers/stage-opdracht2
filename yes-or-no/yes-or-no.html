  • Laat de HTML zoals het is gegeven, doe alles alleen met Javascript en CSS
  • Buttons zijn 100 pixels bij 100 pixels groot
  • Centreer de buttons in de pagina
  • Pas de class "state0" toe op buttons als ze op No staan: state0 geeft als achtergrondkleur: #f88
  • Pas de class "state1" toe op buttons als ze op Yes staan: state1 geeft als achtergrondkleur: #8f8
  • fontgrootte: 20 pixels
  • fontgewicht: vet
  • Werk function clickButton uit en koppel die in een for-loop aan de click-event van de html button-elementen, geen onclick attributen in de HTML
